How the Project Works

This project is a mental health chatbot designed to provide support and information to users. It leverages natural language processing to engage in conversations and direct users to resources based on their input. Users can interact with the chatbot through a friendly interface that responds to various prompts related to mental health topics.

The chatbot is built using modern web technologies including HTML, CSS, and JavaScript. It aims to make mental health resources more accessible and user-friendly for everyone.
